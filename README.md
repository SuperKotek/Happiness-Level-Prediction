# Happiness Score Prediction
## Описание

Happiness Score Prediction — это приложение на языке программирования Python, предназначенное для анализа социально-экономических данных и прогнозирования рейтинга счастья с использованием методов машинного обучения. Проект реализует полный цикл работы с данными: загрузку, предобработку, обучение регрессионных моделей, оценку их качества и визуализацию результатов.

В рамках проекта используются четыре модели регрессии: линейная регрессия, нейронная сеть (MLP), дерево решений и случайный лес. Для удобства взаимодействия с пользователем реализован графический интерфейс, а также телеграм-бот, предоставляющий доступ к функционалу приложения в удалённом формате.

## Основные возможности

- Загрузка данных из Excel-файлов (.xlsx)
- Автоматическая проверка структуры входных данных

Предобработка данных:
- заполнение пропусков средними значениями
- обработка выбросов методом межквартильного размаха (IQR)

Обучение и сравнение четырёх моделей регрессии:
- линейная регрессия
- нейронная сеть (MLP)
- дерево решений
- случайный лес
- Расчёт метрик качества моделей (R², MAE, MSE, RMSE)
- Наглядная визуализация результатов
- Сохранение результатов и метрик

- Работа через графический интерфейс приложения
- Работа через телеграм-бот

## Интерфейс приложения

Приложение состоит из нескольких вкладок, каждая из которых отвечает за отдельный этап работы.

### Вкладка «Загрузка»
- Выбор Excel-файла с данными
- Запуск процесса обработки и обучения моделей
- Отображение статуса выполнения операций
### Вкладка «Исходные данные»
- Таблица с загруженными исходными данными
- Просмотр всех строк и столбцов датасета
### Вкладка «Анализ данных»
- Отчёт о выполненной предобработке
- Информация о заполненных пропусках и обработанных выбросах
- Описательная статистика по признакам
### Вкладка «Результаты моделей»
- График сравнения моделей по коэффициенту детерминации (R²)
- Выбор конкретной модели
- Просмотр метрик качества
- График «Фактические значения vs Предсказания»

## Используемые модели
- Линейная регрессия — базовая модель, описывающая линейную зависимость между признаками и целевой переменной.
- Нейронная сеть (MLP) — многослойный перцептрон, способный моделировать нелинейные зависимости.
- Дерево решений — модель, принимающая решения на основе последовательных условий.
- Случайный лес — ансамблевый метод, объединяющий множество деревьев решений для повышения точности.

## Телеграм-бот

Телеграм-бот расширяет функциональность приложения и позволяет работать с данными без запуска графического интерфейса.

### Возможности бота:

- Приём Excel-файлов с данными
- Автоматическая предобработка и обучение моделей
- Команда /models — сравнение моделей по коэффициенту детерминации (R²)
- Команда /model — выбор одной из четырёх моделей для детального анализа
При выборе модели пользователю выводятся:
- значения метрик качества
- график попаданий (фактические значения vs предсказанные)

### Использование 

Графическое приложение
1. Запустите приложение.
2. Перейдите на вкладку «Загрузка».
3. Выберите Excel-файл с данными.
4. Дождитесь завершения обработки и обучения моделей.
5. Перейдите на другие вкладки для анализа результатов.

Телеграм-бот
1. Запустите бота в Telegram.
2. Отправьте Excel-файл (.xlsx) с корректной структурой.
3. Используйте команды /models или /model для просмотра результатов.
